Triggers

  • Hover off

  • Hover on
  • Hover off

Heart Icon

  • Hover on
  • Hover off

Rules

Asterisk Symbol Button

  • Hover on
    • Asterisk symbol rotates 540 degrees clockwise
    • Asterisk symbol changes color from white to blue
    • Circle shape behind the asterisk symbol changes color from dark gray to light gray
    • A card with product information appears 10px above the Asterisk symbol button with a ease transition
  • Hover off
    • Asterisk symbol rotates 540 degrees counter clockwise
    • Asterisk symbol changes color from blue to white
    • Circle shape behind the asterisk symbol changes color from light gray to dark gray
    • The card with product information (located 10px above the Asterisk symbol button) becomes hidden with a ease transition

Product Information Pop-up Card

  • Hover on
    • 'Add to Cart' Button (under the card) appears with a ease transition
    • 'Asterisk Symbol' Button moves 10px below the 'Add to Cart' Button
  • Hover off
    • 'Add to Cart' Button (under the card) becomes hidden with a ease transition
    • 'Asterisk Symbol' Button moves back to it's original position (10px up from its current position)
    • 'Astrisk Symbol' rotates 540 degrees counter clockwise

Heart Icon

  • Hover on
    • Heart icon expands 1.1x the original scale
    • Heart icon's fill color changes from transparent to red and it doesn't have a stroke/outline
  • Hover off
    • Heart icon returns back to its original scale (1x) with a ease transition
    • Heart icon's fill color changes from red to transparent and it has a black stroke of 1px
