Rules - 'Living room', 'Kitchen & appliances', 'Home office' Button

  • Hover on
    • Cusor changes to a pointer finger
    • The text and arrow to the right of the text (located under the image) changes from a light gray color to a black color
    • The text (beneath the image) is underlined
    • Image is zoomed in about 1.1x the current scale
  • Hover off
    • Cusor changes to default
    • The text and arrow to the right of text changes from a black color to a light gray color
    • The text (beneath the image) is no longer underlined
    • Image returns back to its original scale (1x)
  • Click
    • Links to a new web page
