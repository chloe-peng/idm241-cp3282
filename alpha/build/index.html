Instructions: To find the microinteraction, click on the 'Room' button in the Navigation Menu

Triggers - 'Living Room' Button

  • Hover on
  • Hover off
  • Click

Rules - 'Living Room' Button

  • Hover on
    • Cusor changes to a pointer finger
    • The 'Living Room' text and arrow to the right of the text (located under the image) changes from a light gray color to a black color
    • The 'Living Room' text is underlined
    • Image is zoomed in about 1.1x the current scale
  • Hover off
    • Cusor changes to default
    • The 'Living Room' text and arrow to the right of text changes from a black color to a light gray color
    • The 'Living Room' text is no longer underlined
    • Image returns back to its original scale (1x)
  • Click
    • Links to a new web page
