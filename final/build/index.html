Rules

Asterisk Symbol Button

  • Hover on
    • Asterisk symbol rotates 540 degrees clockwise
    • Asterisk symbol changes color from white to blue
    • Circle shape behind the asterisk symbol changes color from dark gray to light gray
    • A card with product information appears above the Asterisk symbol button with a 0.5s ease transition
    • Cusor changes to a pointer finger
  • Hover off
    • Asterisk symbol rotates 540 degrees counter clockwise
    • Asterisk symbol changes color from blue to white
    • Circle shape behind the asterisk symbol changes color from light gray to dark gray
    • The card with product information (located above the Asterisk symbol button) becomes hidden with a 0.5s ease transition
    • Cusor changes to default
  • Click
    • Links to a new web page

Product Information Pop-up Card

  • Hover on
    • 'Add to Cart' Button (under the card) appears with a 0.5s ease transition
    • Cusor changes to a pointer finger
  • Hover off
    • 'Add to Cart' Button (under the card) becomes hidden with a 0.5s ease transition
    • 'Astrisk Symbol' rotates 540 degrees counter clockwise
    • Cusor changes to default
  • Click
    • Links to a new web page

Heart Icon

  • Hover on
    • Heart icon expands 1.2x the original scale with a 0.5s ease transition
    • Heart icon's stroke color changes from dark gray to red with a 0.5s ease transition
    • Cusor changes to a pointer finger
  • Hover off
    • Heart icon returns back to its original scale (1x) with a 0.5s ease transition
    • Heart icon's stroke color changes from red to dark gray with a 0.5s ease transition
    • Cusor changes to default
  • Click
    • Heart icon's fill color becomes red with a 0.5s ease transition
    • A database is updated adding this item to the user's favorites
  • Click Again
    • Heart icon's fill color becomes transparent

Background - Lamp Photo

  • Hover on
    • Image zooms in 1.1x the original size with a 0.5s ease
    • Asterisk Button appears on top of the lamp
    • Cusor changes to a pointer finger
  • Hover off
    • Image zooms out back to the original size (1x) with a 0.5s ease
    • Asterisk Button is hidden
    • Cusor changes to default
  • Click
    • Links to a new web page

Add to Cart Button

  • Hover on
    • Background colors turns into a darker blue from a lighter blue
    • Arrow appears to the right of the 'Add to Cart' text with a 0.5s ease transition
    • Cusor changes to a pointer finger
  • Hover off
    • Background colors turns into a lighter blue from a darker blue
    • Arrow to the right of the 'Add to Cart' text is hidden with a 0.5s ease transition
    • Cusor changes to a default
  • Click
    • Arrow's vector turns into a checkmark shape
    • The text changes from 'Add to Cart' to 'Added to Cart'
    • A database is updated adding this item to the user's cart
  • Click Again
    • Links to the cart page
